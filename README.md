# Pure HTML Static Site

This folder is a static recreation of the Cullmate site using plain HTML, CSS, and JavaScript.

## Files

- `index.html` - home page
- `product.html` - product and pricing
- `download.html` - download page
- `about.html` - about page
- `faq.html` - FAQ
- `contact.html` - contact page
- `legal.html` - privacy policy and terms
- `changelog.html` - changelog
- `checkout.html` - checkout page
- `styles.css` - shared styling
- `app.js` - shared interactions
- `assets/logo.png` - brand logo

## Deploy

Copy the whole `pure-html/` directory to your web root, or point Nginx/Apache to this folder directly.

If you want clean URLs without the `.html` suffix, add a rewrite rule on the server side later.
